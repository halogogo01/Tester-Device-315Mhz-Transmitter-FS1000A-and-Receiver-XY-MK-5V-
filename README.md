# Tester-Device-315Mhz-Transmitter-FS1000A-and-Receiver-XY-MK-5V-
ฉันสั่งซื้อโมดูลรับส่งสัญญาณ315Mhz และ 433Mhzมาหลายตัว และต้องการอุปกรณ์ทดสอบว่าโมดูลรับส่งสัญญาณ315mhzทั้งหมดนั้นใช้งานได้กี่ตัว มันจึงเป็นที่มาของโปรเจ็คนี้


วัตถุดิบ
1) 315Mhz หริอ 433Mhz Transmitter(Tx) รุ่น FS1000A (โมดูลส่งสัญญาณ)
2) 315Mhz หริอ 433Mhz Receiver(Rx) รุ่น XY-MK-5V (โมดูลรับสัญญาณ)
3) ESP32-C3 Super Mini (บอร์ดไมโครคอนโทรลเลอร์สำหรับเขียนชุดคำสั้ง)
4) LED (หลอดไฟสำหรับแสดงผลเมื่อได้รับสัญญาณ)
5) Push Button Switch (สำหรับกดปุ่มเพื่อออกคำสังให้ส่งสัญญาณ)
6) Perfboard (บอร์ดสำหรับเชื่อมต่อทุกอย่างเเข้าด้วยกัน)
7) Solder (สำหรับประสานรอยต่ออุปกรณ์อิเล็กทรอนิกส์)
8) electrical wire (สำหรับเดินสายเชื่อมต่อในบอร์ด)
9) USB Type C (สำหรับการเชื่อมต่อกับComputer)
10) TP4056 (เผื่อต้องการใช้งานแบบไร้สาย)
11) Battery (เผื่อต้องการใช้งานแบบไร้สาย)


เครื่องมือ
1) soldering iron (สำหรับให้ความร้อนในการประสานรอยของSolder)
2) PCB Hook Knife (สำหรับตัดแผ่นPerfboardให้ได้ขนาด)


ฉันจำลองการเชื่อมต่อPerfboardผ่านโปรแกรม "DIY Layout Creator" (เพื่อจะได้รู้ว่าต้องตัดPerfboardขนาดเท่าไหล)
